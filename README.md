*This is my website. There are many like it, but this one is mine.*

## Setup

``` bash
$ nvm install
$ yarn
```

## Dev
```bash
$ yarn dev # serve with hot reload at localhost:3000
```

## Generate static website
```bash
$ yarn build # output goes to dist/
```
